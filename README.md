### 如何运行项目
1. 运行环境 >= java1.8
2. 初始化数据库 执行init.sql
3. 如果需要使用自己的消息服务器，请网上自行google安装
4. 如果需要使用自己的ES搜索引擎，请网上自行google安装
5. 修改application-dev.yml配置，改为自己的
6. 运行Application
---
### 集成了哪些功能
1. RPC(dubbo服务)
2. 分布式quartz集群
3. 消息中间件rocketMq
4. redis缓存
5. 分布式redis注解锁
6. 自定义注解异常捕获通知钉钉机器人
